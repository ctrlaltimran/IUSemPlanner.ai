# Who is CtrlAltImran?

// developer · student · creator of IUSemPlanner

If you've been using IUSemPlanner.ai to survive Iqra University's course registration, you might be wondering who actually built it.

The tool wasn't built by the university administration or a corporate software company. It was built by a student, for students.

> Name: Imran Murtaza

> Major: BS Software Engineering (Iqra University)
> Portfolio: ctrlaltimran.com

## Why I built IUSemPlanner

Every semester, I watched my friends and classmates struggle with the exact same IULMS portal problems:

  • Manually calculating GPAs across 8 semesters.
  • Accidentally registering for classes that overlap.
  • Getting blindsided by locked prerequisites.

As a Software Engineering student, I realized this was a data parsing problem that code could solve. So, I wrote a custom JavaScript bookmarklet that scrapes the raw IULMS table data, decodes it, and feeds it into a visual planner. No database, no saved passwords—just pure client-side engineering.
